# Scripts

Docker-only scripts. Run from project root.

## Windows (PowerShell)

```powershell
.\scripts\start-dev.ps1    # Dev (watch mode, hot reload)
.\scripts\start-app.ps1    # Prod (production build)
.\scripts\run-tests.ps1    # Tests (unit + E2E)
```

## Linux / Mac / Git Bash

```bash
bash scripts/start-dev.sh    # Dev (watch mode, hot reload)
bash scripts/start-app.sh   # Prod (production build)
bash scripts/run-tests.sh   # Tests (unit + E2E)
```
